Git子模块的删除相对比较麻烦，需要执行多个步骤才能清理干净。

+ 删除子模块目录；
+ 删除 =.gitmodules= 中对应的子模块内容；
+ 删除 =.git/config= 中对应的子模块内容；
+ 删除 =.git/modules/= 下对应的子模块目录；
+ =git rm --cached submodule_name= 。
