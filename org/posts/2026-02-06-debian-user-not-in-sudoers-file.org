为解决 Debian 系统中 ~user not in sudoers file~ 错误，可以参考以下步骤。

首先以 root 用户登录。

#+begin_src bash
su i
#+end_src

打开 ~/etc/sudoers~ 文件，在 ~root ALL=(ALL:ALL) ALL~ 行下添加当前用户的配置。

#+begin_src bash
xxxx    ALL=(ALL:ALL) ALL
#+end_src

保存退出后，注销重新登录，通过以下命令检查是否有 root 权限。

#+begin_src bash
sudo whoami
#+end_src
