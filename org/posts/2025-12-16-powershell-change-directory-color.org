#+TITLE: PowerShell: Change Directory Color
#+AUTHOR: Chuck
#+DESCRIPTION: Use PSStyle to configure how PowerShell render text.
#+KEYWORDS: Themes, PowerShell
#+DATE: <2025-12-16 Tue 17:18>

By default, the directory background color is blue and foreground color is white, it is too shining. We can use [[https://learn.microsoft.com/en-us/dotnet/api/system.management.automation.psstyle?view=powershellsdk-7.4.0][PSStyle]] to configure the color of directory.

Add ~$PSStyle.FileInfo.Directory = $PSStyle.Foreground.Blue + $PSStyle.Bold~ into ~$profile~ file.
